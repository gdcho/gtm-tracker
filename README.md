# gtm-ctm logger

This is a simple logger for Google CTM that logs to the console.

1. Clone the repository

```sh
   git clone https://github.com/gdcho/gtm-tracker
```

2. Load the extension in Chrome by going to `chrome://extensions/` and clicking on `Load unpacked` and selecting the entire folder
3. Go to sites with valid GTM
4. Click on the extension icon and click on the log button
5. Open inspect elements to check logs
